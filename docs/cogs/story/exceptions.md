# File: `cogs/story/exceptions.py`

## Overview
Custom exceptions for the story module. This file is part of the cogs subsystem and handles the primary operations for its respective domain.

## Classes

### `StoryError`
Base exception for story-related errors.

### `StoryNotFoundError`
Raised when a story instance is not found.

### `WorldNotFoundError`
Raised when a world is not found.

### `CharacterNotFoundError`
Raised when a character is not found.

### `StoryDatabaseError`
Raised when there's a database-related error.

### `StoryPromptError`
Raised when there's an error building the story prompt.

### `StoryGenerationError`
Raised when there's an error generating the story response.
