# PigPig Discord Bot Privacy Policy

Effective Date: November 16, 2025
Developer: starpig1129

This Privacy Policy explains how the PigPig Discord Bot (the "Application") collects, uses, and protects your personal information when you use its features. This policy should be read in conjunction with Discord's Privacy Policy.

## 1. Information We Collect

The Application collects the following types of data to provide its 16 core functionality modules:

### 1.1 Discord Interaction Data
- **Basic Discord Data:** Your Discord User ID, Server ID, Channel ID, message timestamps, and server membership information
- **Conversation Content:** All text content of your conversations with the Application in servers and direct messages
- **Voice State Data:** Voice channel join/leave events and voice state changes for music functionality

### 1.2 AI-Driven Communication Data
- **AI Conversation Memory:** The Application uses its "Episodic Memory System" to permanently store conversation history via semantic search for enhanced conversational context and memory features
- **System Prompt Data:** Custom system prompt configurations and personality settings stored per server
- **Story and Narrative Data:** Interactive story content, user choices, and narrative progress for the Story Management module

### 1.3 Multi-Modal Content Data
- **Image Content:** Image content you upload for Visual Question Answering (VQA), image analysis, and AI image generation/editing
- **Media Files:** GIF files, multimedia content processed through various tools (GIF Tools, Image Generation)

### 1.4 Functional Preference Data
- **User Data Management:** Personal user profiles, preferences, and background information stored through the User Data system
- **Music Preferences:** Playback history, queue preferences, and listening habits for music functionality
- **Channel Configuration:** Server-specific settings, channel modes (whitelist/blacklist), and custom response configurations
- **Reminder and Schedule Data:** Personal reminders, scheduling information, and calendar data uploaded via the Schedule Manager

### 1.5 Service Interaction Data
- **Search Queries:** Internet search history, restaurant recommendation ratings ("Eat" function), and feedback data
- **Mathematical Calculations:** Mathematical expressions and calculation history for the Math Calculator module
- **Language Preferences:** Server language settings and localization preferences
- **Update and System Data:** Bot update logs, system status information, and operational metrics

### 1.6 Administrative Data
- **Bot Configuration:** Bot Owner ID, administrative permissions, bug report channel settings
- **Error Reports:** System error logs and diagnostic information for maintenance purposes

## 3. Data Sharing and Storage (Third-Party Services)

As the Application is driven by multiple Large Language Models (LLMs) and external services, your interaction data will be shared with third-party service providers:

### 3.1 AI and Language Model Services
To provide AI responses and multi-modal functions, your **conversation content, image content, voice data, and interaction metadata** will be transmitted to the following **Third-Party API Services** for processing:

- **Anthropic (Claude):** For advanced conversational AI and reasoning capabilities
- **OpenAI (GPT):** For language understanding, generation, and analysis
- **Google (Gemini):** For multi-modal AI, image analysis, and grounding capabilities
- **Local AI Models:** For privacy-sensitive operations and offline functionality

**Important:** These services have their own independent Terms of Service and Privacy Policies. You should review their respective policies.

### 3.2 Content and Media Services
- **YouTube API:** For music streaming, search, and metadata retrieval
- **Google Search API:** For internet search functionality
- **GIF Services:** For GIF search and retrieval operations

### 3.3 Data Storage and Processing
- **Primary Storage:** Data is stored on Virtual Private Servers (VPS) with appropriate security measures
- **Backup Systems:** Regular backups maintained for data protection and recovery
- **Data Location:** Server infrastructure may be located in various jurisdictions for optimal performance

## 4. Data Retention and Your Rights

### 4.1 Data Retention Policy
- **Long-Term Memory:** Your conversation memory data is **retained indefinitely** until actively deleted by you or a server administrator using bot commands
- **User Data:** Personal user information is maintained as long as the Application remains active in your server
- **Temporary Data:** Search queries, calculations, and temporary processing data are typically retained for 30 days unless needed for troubleshooting
- **Deletion Triggers:** Data is automatically deleted when the Application is removed from the server or through explicit user commands

### 4.2 Your Rights
You have the right to:
- **Access:** Request access to personal data stored by the Application
- **Correction:** Request correction of inaccurate personal information
- **Deletion:** Request deletion of your personal data
- **Portability:** Request a copy of your data in a structured format
- **Restriction:** Request restriction of processing in certain circumstances

### 4.3 Request Process
To exercise any of these rights, please contact us using the "Contact Information" provided below. We will respond to legitimate requests within 30 days.
