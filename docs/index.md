# PigPig Discord Bot 說明文檔

歡迎使用 **PigPig Discord Bot** 的官方說明文檔！

這是一個基於先進 AGI 與認知架構設計的 Discord AI 機器人，專注於強化學習、認知心理學與神經符號 AI 的整合。

## 專案特性
- **模組化認知 (Modular Cognition)**: 分離感知、記憶、推理與行動。
- **強大的記憶系統**: 包含情節記憶與工作記憶管理。
- **多模型支援**: 整合 Google Gemini, OpenAI, Claude 等多種 LLM。
- **豐富的功能模組**: 具備搜尋、繪圖、音樂、數學與系統管理功能。

## 快速開始
您可以參考 [GitHub README](https://github.com/starpig1129/ai-discord-bot-PigPig) 獲取安裝與部署指南。
